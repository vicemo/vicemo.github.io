Fuzzy Homologation System

Enterprise application built to automate product homologation using fuzzy matching algorithms (Jaro-Winkler and Soundex). It reduces manual effort and ensures data consistency across systems.

Tech Stack: ASP.NET MVC, C#, SQL Server, jQuery, DataTables

View Code on GitHub

Main Workflow

Main interface where users can perform auto match, manual search, assign homologations, skip records or mark them as not found.
Automatic matching process using fuzzy matching algorithms to suggest optimal data matches.
Manual search mode allowing the user to validate and select the correct homologation.
View of successfully homologated records, ensuring consistency across systems.
Records categorized as "Not Found" for further validation or analysis.

Role-Based Access & Administration

Admin dashboard for managing users and assigning roles.
Editing user roles with ADMIN and REVIEWER permissions.

Architecture & Security

IIS configuration with Windows Authentication (SSO) for secure and seamless user login.
Backend implementation using ASP.NET MVC and SQL Server integration.
